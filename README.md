# Pipeline Datos de Financieros
Desarrollo de una solución integral para la extracción automatizada de valores liquidativos (NAV) vía Web Scraping. El sistema valida y consolida datos históricos en Excel, ejecuta la actualización mediante scripts .bat y visualiza la evolución financiera en un dashboard interactivo. (Python, Pandas, Excel, Power BI, Batch Scripting).
